# synonym_cilin

## 介绍

**synonym_cilin**是一种基于层次结构的同义词林同义词、同类词查询方法。

基于 [Final_word_Similarity](https://github.com/yaleimeng/Final_word_Similarity/blob/master/cilin/V2/ciLin.py)
V2版的同义词词林词语相似度计算方法。在 [@yaleimeng](https://github.com/yaleimeng) 基础上加入字典树结构，用于查询层次结构下的同义词和同类词。

代码示例：

```python
sc = synonym_cilin.SynonymCilin("data/cilin.txt")

# 同义词查询
res = sc.get_synonym_words('苹果')
print(res)

# 同类词（相似词）查询
res = sc.get_similar_words('苹果', min_sim=0.9)
print(res)
res = sc.get_similar_words('野生', n=6)
print(res)
```
